# 💼 Job Portal Web Application

## 📌 Project Overview

The Job Portal Web Application is a full-stack web application developed for students and recruiters. It allows students to browse jobs, save jobs, and apply for jobs, while recruiters can post and manage job openings.

This project is built using Spring Boot for the backend, MySQL as the database, and HTML, CSS, and JavaScript for the frontend.

---

## 🚀 Features

### Student
- User Registration
- User Login
- View Available Jobs
- Search Jobs
- Save Jobs
- Apply for Jobs
- View Applied Jobs

### Recruiter
- Add Job Openings
- Update Job Details
- Delete Jobs
- View Applications

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/kaviyadharmaraj6369/JobPortal.git
```

### 2. Open in IntelliJ IDEA

Import the project as a Maven project.

### 3. Configure MySQL

Create a database:

```sql
CREATE DATABASE jobportal;
```

Update the database configuration in:

```
src/main/resources/application.properties
```

### 4. Run Project

Run

```
JobPortalApplication.java
```

Open:

```
http://localhost:8080
```
